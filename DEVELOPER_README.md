# Developers

## Pre-requisites
- NodeJs installed
- ```$ npm install -g angular-cli```

## Build
```
$ git clone https://github.com/BrightChain/quri.git
$ cd quri
$ npm install
$ npm run build:development
```

## Run Tests
```
$ npm run test
```

## Serve Locally
```
$ npm run start
# open browser as directed
```
